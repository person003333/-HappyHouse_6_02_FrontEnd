# 페이지 설계
## 1. 메인 페이지 😎
### 페이지 설명
     로그인 하지 않았을 경우 메인 로고와 동별/아파트별 실거래가 검색 과 지도가 나타난다.
     로그인 하였을 경우 추가적으로 로그아웃, 회원 정보, 관심지역 등록 버튼이 나타난다.
